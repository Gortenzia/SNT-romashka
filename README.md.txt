# SNT - Romashka
![Logo](https://github.com/Gortenzia/SNT/blob/master/logo/ромашка%20на%20зеленом%20фоне.png?raw=true)

## Первый проект Frontend-части сайта, родом из 2020 года

**СНТ-Ромашка** — СНТ, состоящее всего из шестидесяти участков затерявшееся на окраинах Павловского-Посада в Московской области. 
Входит в состав больших Улитинских дач. 
Даже маленькое СНТ заслуживает своего сайта.

**На сайте будет находиться:**
* Протокол последних собраний
* Сервис по поиску задолженностей, написанный на JavaScript`e
* Схема проезда
* Небольшой дачный форум
* Все необходимые контакты
